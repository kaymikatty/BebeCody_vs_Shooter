# BebeCody_vs_Shooter

Para poder ejecutar BebeCody vs Shooter se debera tener previamnete instalado lo siguiente:

  - Libreria pygame
  - Módulo Os
  
Y para poder jugar se deberá iniciar el archivo main.py
